## GitHub Pages 是什么

[GitHub Pages](https://pages.github.com/) 是一个由 GitHub 提供的免费静态网站托管服务，允许用户直接从 GitHub 仓库中发布个人或项目的网页。用户可以通过编辑、推送更改，使网站立即生效。这些网站可以用于展示个人简历、项目文档、博客等内容，无需复杂的服务器配置或数据库支持。
